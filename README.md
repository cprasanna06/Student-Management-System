# Student Management System

A Java console-based application developed using Object-Oriented Programming principles to manage student records.

## Features

- Add Student
- View All Students
- Search Student by ID
- Update Student Details
- Delete Student
- Exit Menu

## Concepts Used

- Java
- Classes & Objects
- Encapsulation
- Constructors
- Arrays
- Methods
- Loops
- Conditional Statements

## Project Structure

- Student.java
- StudentManagement.java

## Future Improvements

- Store data using files
- Connect with MySQL database
- Java Swing GUI
- JavaFX GUI
